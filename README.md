# 🤖 RAG Chatbot using LangChain + Streamlit

An end-to-end Retrieval-Augmented Generation (RAG) chatbot that allows users to upload documents and ask questions based on the content.

---

## 🚀 Features

- 📄 Upload PDF documents
- 🔍 Context-aware semantic search using vector database
- 🤖 Conversational AI powered by LLM
- 🧠 Memory-enabled chat experience
- ⚡ Fast retrieval using ChromaDB
- 🌐 Interactive UI using Streamlit

---

## 🛠️ Tech Stack

- Frontend: Streamlit
- Backend: Python
- LLM Framework: LangChain
- Vector DB: ChromaDB
- Embeddings: Sentence Transformers
- API: OpenAI
